# lenaa Ads API

Lokale Node-API als deterministischer Zwischenlayer für Ads-Daten.

## Start

```bash
cd /home/hermes/.hermes/profiles/lenaamarketing/apis/ads-api
npm start
```

## Endpoints

- `GET /health`
- `GET /request-test` — sendet einen ausgehenden Request an `TEST_REQUEST_URL`

## Config

Siehe `.env.example`. Keine echten Secrets committen.
